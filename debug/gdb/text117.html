KGTP

  • KGTP is a Linux kernel GDB debugger and tracer that uses Kprobe.

    • It provides a tracepoint interface for GDB to connect to via debugfs

  • Quick Start

    • Download http://code.google.com/p/kgtp/downloads/list
    • 'make' to compile
    • sudo insmod gtp.ko
    • sudo gdb ./vmlinux (gdb) target remote /sys/kernel/debug/gtp
