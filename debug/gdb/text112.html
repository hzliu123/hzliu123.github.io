Analyze Collected Data

  • tfind [n | start | end | -]

    • Select a trace frame; all gdb commands (print, info registers, backtrace, etc.) will begin to operate on the data collected in the frame

  • tfind tracepoint [num]

  • tfind pc [addr]

  • tfind outside addr1, addr2

  • tfind range addr1, addr2

  • tfind line [[file:]n]
