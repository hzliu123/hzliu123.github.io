Commands to Specify Files

  • file [file]

    • Use file for both symbols and executable; with no arg, discard both

  • core-file(core) [file]

    • Read file as core dump; or discard

  • exec-file [file]

    • Use file as executable only; or discard

  • symbol-file [file]

    • Use symbol table from file; or discard
