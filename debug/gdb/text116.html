Using Trace Files

  • You can dump the trace data into a file, and later use that file as a source of trace data

  • tsave [ -r ] filename

    • Save the trace data to filename on the host filesystem (or target filesystem if -r is specified)

  • target tfile filename

    • Use a trace file filename as a target
    • filename must be on a filesystem accessible to the host
